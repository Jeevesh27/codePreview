Traverse nodes level by level using a queue.
Speed
Pseudocode
1  LEVELORDER(root):
2    if root == NULL: return
3    create queue Q
4    enqueue(root)
5    while Q not empty:
6      node = dequeue(Q)
7      visit(node)
8      if node.left != NULL: enqueue(node.left)
9      if node.right != NULL: enqueue(node.right)
